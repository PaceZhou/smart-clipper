# SmartClipper Chrome Extension

## 功能

- 🎨 三种阅读模式（明亮/深色/护眼）
- ⌨️ 快捷键切换（Ctrl+Shift+R / Cmd+Shift+R）
- 📖 Safari 风格排版
- 🖼️ 不重新加载，保留原图片

## 安装

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `extension` 文件夹

## 使用

- **快捷键**: `Ctrl+Shift+R` (Win) / `Cmd+Shift+R` (Mac)
- **切换主题**: 点击扩展图标选择主题

## 特点

✅ 在原网页直接注入样式，不重新加载  
✅ 图片不会失效（微信公众号等）  
✅ 即时切换主题  
✅ 轻量级，无需服务器
