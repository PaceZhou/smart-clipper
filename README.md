# SmartClipper

Universal web clipper with Notion integration - Support WeChat articles and general web pages.

## 🎯 Features

- 🌐 Universal web page scraping
- 📱 WeChat article specialized support
- 📝 Notion database integration
- 🔄 Cross-device sync via Chrome extension
- 🖥️ Desktop application (coming soon)
- 📱 Mobile apps (future)

## 🏗️ Architecture

```
smart-clipper/
├── core/           # Core scraping & export engine
│   ├── scrapers/   # Web scrapers (WeChat, Universal)
│   ├── exporters/  # Export to Notion, Markdown, etc.
│   └── utils/      # Shared utilities
├── extension/      # Chrome extension
├── desktop/        # Electron desktop app (future)
└── server/         # Processing server
```
